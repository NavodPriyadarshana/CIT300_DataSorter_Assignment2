## üìñ Project Overview
This Java program compares the performance of three common sorting algorithms ‚Äî **Bubble Sort**, **Merge Sort**, and **Quick Sort** ‚Äî by measuring:
- Execution time (in milliseconds)
- Number of comparison operations performed  

It uses a **menu-driven interface** that allows users to:
1. Enter numbers manually or generate random data  
2. Perform each sorting algorithm individually  
3. View a **comparison table** of all algorithms  

The system was developed collaboratively using **GitHub branches and pull requests**.
